**Barbearia Estilo Fino**

Interface inicial desenvolvida com HTML, TailwindCSS e boas práticas de design responsivo

**Sobre o Projeto**

Este projeto foi desenvolvido como parte das atividades do módulo Desenvolvimento Web com Frameworks e HTML/CSS da UFMS Digital.
O objetivo é construir uma interface dinâmica e moderna utilizando um framework CSS — no caso, Tailwind CSS — aplicando conceitos de responsividade, organização e estilização limpa.

A página representa a tela inicial de uma barbearia fictícia chamada Barbearia Estilo Fino, com design elegante, visual profissional e foco em experiência do usuário.

**Tecnologias Utilizadas**

- HTML5
- Tailwind CSS (via CDN)
- Google Fonts (Inter)
- Sombras, transições e animações CSS
- Layout totalmente responsivo

**Destaques do Projeto**

- ✔ Layout moderno e minimalista
- ✔ Uso eficiente do Tailwind para estilização rápida
- ✔ Imagens com efeito de destaque (hover lift)
- ✔ Botões com animação suave
- ✔ Responsividade completa (mobile, tablet e desktop)
- ✔ Tipografia profissional
- ✔ Paleta escura e sofisticada inspirada em barbearias premium
